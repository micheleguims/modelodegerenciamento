# 🚀 SME Admin Portal - Piloto de Modernização Tecnológica

Este projeto é um "Prova de Conceito" (PoC) desenvolvido para a Secretaria Municipal de Educação (SME), com o objetivo de validar uma arquitetura de alta performance e baixo custo como alternativa ao uso do MS PowerApps.

## 🎯 Objetivo Estratégico

Testar a viabilidade de **Web Apps dedicados** (React) na substituição de fluxos complexos, garantindo:
- **Alta performance com custos controlados**: Validação de uma estrutura escalável.
- **Maior fluidez e velocidade**: Eliminação de barreiras de login e lentidão observadas em ferramentas low-code.
- **Independência técnica**: Capacidade de integrar Inteligência Artificial (ex: Gemini API) e rodar em servidores internos.

## 🛠️ Stack Tecnológica

O projeto utiliza um ecossistema moderno focado em agilidade e segurança:

- **Front-end**: React + Tailwind CSS (Identidade visual baseada nas cores corporativas #13335a e #66b6e3).
- **Backend/Database**: Firebase Firestore (Persistência em tempo real e suporte offline).
- **Gráficos**: Recharts para monitoramento de métricas.
- **Ícones**: Lucide React.

## ✨ Funcionalidades Principais

- **Painel de Controle**: Métricas consolidadas e volume de registros por CRE.
- **Base de Dados Corporativa**: Gestão de documentos (PDF e Planilhas) com controle de status e anexos.
- **Trilha de Auditoria Segura**: Registro completo de ações (Logs) para garantir a rastreabilidade (Nível Central).
- **Sobre o Piloto**: Aba institucional detalhando a arquitetura, custos de escala (Blaze) e estratégia de transição.

## 💻 Instalação e Uso Local

1. Clone o repositório:
    git clone [https://github.com/micheleguims/modelodeuso.git](https://github.com/micheleguims/modelodeuso.git)

2. Instale as dependências:
    npm install

3. Configure o Firebase:
    Crie um arquivo .env na raiz com suas chaves de API (verifique o .gitignore).

4. Inicie o projeto:
    npm start
